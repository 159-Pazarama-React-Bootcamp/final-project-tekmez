## Bitirme projesi
### Başvuru / ticket yönetim sistemi

### Özellikler

------------


- Başvuru formu oluşturma
- Oluşuturulan forma özel numara ile durum sorgulama
- Başvuruyu red veya onaylama
- Başvuruya özel cevap yazma

### Kullanılan Teknolojiler & Araçlar

------------

- Javascript
- React
- React Router
- Redux Toolkit
- Babel
- Eslint
- MockApi
- Netlify
- Jest
